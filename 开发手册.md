# xhs-dl 开发与发布手册

## 1. 产品边界

xhs-dl 只处理用户有权保存的公开小红书笔记。默认直接解析公开页面并在本地保存，不要求登录，不把链接转交给第三方解析站；不绕过私密、删除、地区、年龄或其他访问限制，也不移除创作者已经嵌入媒体的署名。

## 2. 架构与数据

- Windows：`xhs_dl/desktop/app.py` 提供桌面界面，`xhs_dl/core/` 负责解析和下载。
- Android：`android/app/` 是原生应用，公开页解析、MediaStore 保存、历史记录均在设备本地完成。
- 每条笔记目录只允许媒体文件和 `文案.txt`；正文文件包含标题、正文、话题。
- Windows 历史统一放在 `%LOCALAPPDATA%\xhs-dl\history.json`；Android 历史放在应用内部数据目录。单条历史只保留下载网址、笔记 ID、标题。
- 目录命名为 `评数字-赞数字-标题-作者`，无法取得互动数时使用“未知”。

## 3. 视觉语言（发布阻断项）

正式视觉只有两套：默认“拟态悬浮（neo）”和备用“克制玻璃（glass）”。两套都使用低饱和冷灰蓝、柔和层次、清晰中文和一个主操作，不使用霓虹紫蓝、通用 AI 后台、纯白卡片墙或装饰性假控件。

- Windows、Web、Android 的名称、颜色关系、圆角、主次按钮和状态表达必须统一。
- 主题选择必须持久保存，切换不能清空用户输入或设置。
- 主界面、设置弹窗、历史弹窗、错误提示必须属于同一主题体系。
- 所有底部主操作必须始终可见；内容变高时使用滚动或可调整窗口，不能依赖固定小高度。

详细色值见 `skills/xhs-download/references/visual-language.md`。

## 4. Windows 高 DPI 验收

每次桌面 UI 改动至少检查 100%、125%、150%、175% 显示缩放：

1. 主窗口标题、主题、设置按钮和底部操作完整显示。
2. 打开设置，确认“保存设置”无需拖动或猜测即可看到并点击。
3. 切换两次主题，确认 neo 与 glass 都正确显示，输入内容不丢失，重启后主题仍保留。
4. 检查 1280×720 与常用高分屏；窗口过小时允许调整大小，不允许裁掉主按钮。
5. 截图必须使用 DPI 感知坐标，避免把错误截图当成界面裁切。

## 5. Android 验收

1. 在真实授权设备安装 APK 并启动，不以“编译成功”代替真机成功。
2. 检查竖屏滚动、系统字体放大、设置弹窗、两套主题和底部操作。
3. 通过分享文本和手动粘贴两条入口测试链接识别。
4. 用一条真实公开笔记下载，核对 `Download/xhs-dl/评-赞-标题-作者/` 中媒体与 `文案.txt`。
5. 检查内部历史记录、失败提示、进度、屏幕常亮释放和应用重启。

## 6. 版本、打包与发布

- Windows 修复版本同步更新 `pyproject.toml`、`xhs_dl/__init__.py`、CLI、Web 页脚、手册和变更日志。
- Android 版本同步更新 `versionCode`、`versionName`、界面页脚和更新比较版本。
- Windows 运行测试后执行 `build-portable.ps1`，再从最终 EXE 做一次启动与真实下载。
- Android 先构建 debug、真机安装验证，再构建 release APK。
- 发布前核对：测试、安装包、说明、变更日志、共享技能包、Git 提交与推送、GitHub Release 附件、校验值、本地快捷方式。
- Windows 标签使用 `vX.Y.Z`；Android 标签使用 `android-vX.Y.Z`，避免互相干扰更新检测。

## 7. 诊断、备份与恢复

- 先保留失败提示、HTTP 状态、笔记 ID 和阶段信息，日志不得记录 Cookie、令牌或私人笔记内容。
- 发布版本和 Git 标签是云端可恢复备份；上一稳定版不得覆盖或删除。
- 升级不清空用户下载目录、设置和集中历史。发生回归时可把桌面快捷方式改回上一稳定版 EXE。

## 8. 已沉淀的回归案例

### V2.3.0 桌面主题缺失

原因：从 Web 迁移到独立桌面时继承了颜色，但“双主题切换与持久化”没有进入桌面验收清单。修复：桌面显式提供 neo/glass 调色板、切换按钮和设置持久化；发布前必须切换、重启验证。

### V2.3.0 设置按钮被遮挡

原因：设置弹窗使用固定较小高度，未覆盖 Windows 高 DPI 缩放。修复：增大默认尺寸、允许调整、保留最小尺寸和底部安全间距；把 100%–175% DPI 与 1280×720 检查列为发布阻断项。
