# 小红书无水印下载器 V2.3 便携版

## 设计说明

桌面版面向普通 Windows 用户，主流程只有“粘贴—识别—采集”。视觉延续低饱和冷灰蓝和克制的轻拟物风格；保存位置、频率和更新检测集中在右上角“设置”，避免干扰主操作。

## 首次使用

1. 如果本机尚未安装下载引擎，双击 `install-engine.bat` 并等待完成。
2. 双击 `xhs-dl.exe`。
3. 手动粘贴分享内容后点“开始采集”，或者复制小红书链接后点“粘贴并采集”。

默认保存到当前用户的 `Downloads`。点击“设置”可选择其他文件夹。

## 自动频率

- 1 条：直接采集。
- 2–20 条：每篇随机等待 35–55 秒。
- 21–50 条：每篇随机等待 55–85 秒。
- 50 条以上：每篇随机等待 110–160 秒。

设置中可以手动选择其他档位。随机慢速只能降低机械高频请求风险，不能保证平台永不触发访问限制。

## 下载结果

文件夹格式：`评128-赞3560-帖子标题-作者`。

每篇笔记文件夹只包含图片或视频以及 `文案.txt`。文案文件包含标题、正文和话题，不放 JSON 或数据库。

所有成功历史集中保存在 `%LOCALAPPDATA%\xhs-dl\history.json`，每条只有下载网址、笔记 ID、标题三个字段。软件设置保存在同目录的 `settings.json`。

## 更新与安全

软件默认在启动后检查 GitHub Releases 版本号，发现更新时显示入口，但不会静默安装。公开笔记无需密码或 Cookie；软件不绕过私密、删除、地区或其他访问限制，也不擦除作者嵌入原图的署名。
