## 核心能力

- 公开笔记、免账号登录、原始图片/视频本地保存。
- 短链接、长链接、整段分享文本、批量去重与分速率执行。
- CLI、浏览器可视化界面、AI/OpenClaw 共享技能。
- 每篇结果与批次进度均写入 JSON 清单。
- Web 后台任务、逐条进度、拟态悬浮与克制玻璃双主题。

## 边界

- 仅处理用户有权保存的公开内容。
- 不移除创作者嵌入媒体本身的署名。
- 不绕过私密、删除、地区、年龄或其他访问限制。
- 默认不把链接发送给第三方在线解析网站。

## 发布核对

测试通过后提交并推送源码，生成 Windows 源码/启动包与独立技能包，计算校验值，创建 GitHub Release 并验证附件可见。
