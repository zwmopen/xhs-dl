## 核心能力

- 公开笔记、免账号登录、原始图片/视频本地保存。
- 短链接、长链接、整段分享文本、批量去重与分速率执行。
- CLI、浏览器界面、Windows、Android、iPhone 三套独立客户端，以及 AI/OpenClaw 共享技能。
- 三端均可设置下载目录；不做设备互联或云同步。
- 笔记目录只放媒体与文案.txt；精简历史集中保存到应用数据目录的 history.json。
- 设置按钮、文件夹选择、剪贴板识别、自动频率判断、更新检测与逐条进度。

## 边界

- 仅处理用户有权保存的公开内容。
- 不移除创作者嵌入媒体本身的署名。
- 不绕过私密、删除、地区、年龄或其他访问限制。
- 默认不把链接发送给第三方在线解析网站。

## 发布核对

测试通过后提交并推送源码，生成 Windows 源码/启动包与独立技能包，计算校验值，创建 GitHub Release 并验证附件可见。
