# 安全与注意事项

1. 仅处理你提供或授权的公开笔记，不绕过私密、删除、年龄、地区或访问限制。
2. 正常公开笔记下载不需要登录，不要向任何人或脚本提供小红书密码、Cookie、浏览器数据。
3. 默认只使用本地引擎。在线去水印网站会获知你的链接和兴趣信息，软件不会静默调用。
4. 下载内容仍属于原作者；无平台水印不代表获得转载、商用或再发布授权。
5. 创作者主动做进图片的署名、Logo 和品牌标识属于原始内容，不会移除。
6. 批量下载保持 `cautious` 或 `slow`，避免频繁请求触发风控。
7. 第三方引擎固定到经过实测的版本后再发布；升级时先审查来源并重新跑真实样本与自动测试。
8. 下载结果默认只保存在本机，不上传 GitHub Release 或公共云盘。
